# Event
Pengaturan event yang diadakan oleh penyelenggara.  
Kapan, waktu, tempat, jumlah peserta


## Commands

#### EventCreate  
membuat sebuah event.

## EventUpdate
memperbahuri event yang telah dibuat.  
Event yang sudah terjadi, tidak dapat di ubah.

## EventDelete
Menghapus event
